Financial Portfolio Management System
Welcome to the Financial Portfolio Management System, a comprehensive platform designed to assist users in managing their investment portfolios efficiently.

Getting Started
To use the Financial Portfolio Management System, follow these steps:

Clone the repository to your local machine.
Install the necessary dependencies.
Set up the database and configure the connection.
Run the application.

User Authentication
Upon running the application, you will be greeted with the message "Welcome to the Financial Portfolio Management System." You will then be presented with two options:
1. Login: If you are an existing user, enter your username and password to log in.
2. Register: If you are a new user, enter your desired username, email address, and password to register. Once registered, you will be prompted to log in.

Main Menu
After successfully logging in, you will be presented with a menu of options:
1. Securities
2. Watchlist
3. Portfolio
4. My Account
5. Logout

Securities
If you choose option 1, you will see a list of all available securities. Enter the ID of the security you wish to view. You will then see detailed information about the selected security, including its market price, historical data, and options to add it to your watchlist, buy, or sell.

Watchlist
Selecting option 2 will display all securities in your watchlist. You can choose to select a security to view its details, delete a security from your watchlist, or return to the main menu.

Portfolio
Option 3 allows you to view all portfolios you have created. You can select a portfolio to view its contents, including the securities it contains. From there, you can choose to sell securities within the portfolio or return to the main menu.

My Account
Selecting option 4 will give you access to various account management functions:
1. Check Balance: View the balance of your investment account.
2. Deposit: Deposit funds into your account.
3. Withdraw: Withdraw funds from your account.
4. Search Transaction: View transaction history and details.

Logout
Selecting option 5 will log you out of the Financial Portfolio Management System, returning you to the login/register screen.
